# Commandes Git essentielles pour VestiaireVert

## Commandes quotidiennes
# Vérifier l'état des fichiers
git status                  # Voir les fichiers modifiés/ajoutés/supprimés

# Ajouter des modifications
git add .                  # Ajouter tous les fichiers modifiés

# Créer un commit
git commit -m "message"    # Créer un commit avec un message

# Pousser les modifications
git push     # Envoyer les commits vers GitHub

## Récupérer les modifications
git pull      # Récupérer et fusionner les modifications depuis GitHub
